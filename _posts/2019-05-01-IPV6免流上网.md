本篇主要介绍怎么利用IPV6免流上网，下面看看大概的思路：

1、测试自己所处网络环境有没有IPV6

2、获取学生包或者用钱购买VPS，推荐digitalocean或者vultr或者搬瓦工，并创建实例

3、windows环境下用xshell或者putty连接云服务器，用脚本搭建SS/SSR

4、下载SS/SSR客户端，配置模式，进行科学上网，更高要求，可使用proxifier

接下来就是用别人的一键脚本安装SS，（直接在绿色光标处复制该行命令回车即可，只需要执行一次，卸载ss后也不需要重新下载），这是我看的教程：[https://www.flyzy2005.com/fan-qiang/shadowsocks/install-shadowsocks-in-one-command/](https://www.flyzy2005.com/fan-qiang/shadowsocks/install-shadowsocks-in-one-command/)

> 先安装git
> Centos执行这个： `yum -y install git`
> Ubuntu/Debian执行这个： `apt-get update && apt-get -y install git`
> 然后 执行这行命令：`git clone https://github.com/flyzy2005/ss-fly`
> 完成后再运行搭建SS脚本代码：`ss-fly/ss-fly.sh -i flyzy2005.com 1024`

其中flyzy2005.com换成你要设置的shadowsocks的密码即可（这个flyzy2005.com就是你ss的密码了，是需要填在客户端的密码那一栏的），密码随便设置，最好只包含字母+数字，一些特殊字符可能会导致冲突。而第二个参数1024是端口号，也可以不加，不加默认是1024~（举个例子，脚本命令可以是ss-fly/ss-fly.sh -i qwerasd，也可以是ss-fly/ss-fly.sh -i qwerasd 8585，后者指定了服务器端口为8585，前者则是默认的端口号1024，两个命令设置的ss密码都是qwerasd）

这个看上面的教程就能完成，我就不多说了。

搭建好后需要修改一下ss配置文件，一般ss的配置文件是这样的（配置文件路径在 /etc/shadowsocks.json）：
{    "server":"0.0.0.0",    "server_port":1024,    "local_address": "127.0.0.1",    "local_port":1080,    "password":"mypassword",    "timeout":300,    "method":"aes-256-cfb"}

将0.0.0.0改成::即可，即：

{    "server":"::",    "server_port":1024,    "local_address": "127.0.0.1",    "local_port":1080,    "password":"mypassword",    "timeout":300,    "method":"aes-256-cfb"}

::的意思是同时监听IPv4和IPv6。之后重启ss：ssserver -c /etc/shadowsocks.json -d restart。至此，服务器就支持ss的IPv6访问了。 最好在等一下的客户端里把地址设置成ipv6地址。

之后就可以下载[各种平台版本的shadowsocks](https://github.com/shadowsocks)了 。下面是SS的截图
![SS](https://s2.ax1x.com/2020/01/19/1CdnZn.png)
把你在xshell里面搭建时的端口和密码，地址填入里面，确定。然后右击小飞机图标，点击最上面的启用系统代理，再点击系统代理模式，里面有两个模式，全局模式，就是全部走服务器代理；还有PAC模式，就是指国内可以访问的站点直接访问，不能直接访问的再走shadowsocks代理~。一般情况下，SSR里面的全局并不能实现所有的全局，因为[我用这个IP地址测试网站](https://www.ip138.com/), 测试还是显示我当地的地址，SS开了全局模式就是显示服务器的地方。当然无论SSR或者SS，要想实现其他应用程序的代理，要结合[proxifier](https://www.proxifier.com/)。关于这方面的教程，既然你已经可以科学上网了，那就去油管找吧，里面还是有很多的，还是在[这里推荐一个吧](https://www.proxifier.com/)。好了，现在就能实现IPV6免流了，看我把网关断了，依然能连上百度。
